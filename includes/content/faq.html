Frequently Asked Questions

What's a "whois lookup"?

It's a database record lookup to see who own a certain domain, when was it registered and so on.

This is also a way to check for domains that are not registered yet - if there's no record, you can register this domain yourself.

Wikipedia has a more detailed answer.

How does this site doing the lookups

This site is simply a web interface to run the unix `whois` command.
